loginPostController(req, res, next)

This is the controller for handling login request. It will validate if the user's input is relevant with the data saved in the databse. And then will fetch the user's credentials from the databse to check if the user's credential is matched with the given input. If the request is succesfull it will redirect the User to respective pages based on their roles. And if any errors occures, it will be handled by global error handler.
